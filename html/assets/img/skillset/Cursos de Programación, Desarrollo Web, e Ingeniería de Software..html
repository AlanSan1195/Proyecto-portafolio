Hola Oliver. Para mostrar imágenes localmente en tu sitio web, necesitas asegurarte de que la ruta de archivo que estás usando es correcta y está en el lugar adecuado.

Asegúrate de que las imágenes se encuentran en una carpeta en la misma ubicación que tus archivos HTML y CSS. Además, es importante recordar que las rutas de archivo son sensibles a mayúsculas y minúsculas, por lo que debes asegurarte de que estás escribiendo la ruta exactamente como se encuentra en tu sistema de archivos.

Si sigues teniendo problemas para mostrar imágenes locales, también puedes intentar abrir el archivo de imagen directamente en tu navegador para asegurarte de que la imagen se cargue correctamente y esté en la ubicación adecuada. Para mi ejemplo uso la extensión de Liver Server que crea un servidor que sirve las imágenes. Esto lo explico en la descripción de la lección de imágenes.

Espero que esto te ayude a resolver tu problema. Dime como te va con eso!

Xavier Reyes Ochoa
en 15 de abr

Hola Gerardo,

Buena idea en reemplazar la sección si estás empezando pero no veo ningún problema tampoco si lo dejas como experiencia e incluyes tu experiencia estudiando ya que yo lo hice como lo muestro cuando estaba empezando con experiencia en proyectos personales o para amigos.

Para hacer que las imágenes ocupen todo el campo, puedes utilizar la propiedad CSS background-size en la clase CSS que estés utilizando para la zona de formación. Puedes establecer background-size: cover; para hacer que la imagen cubra todo el contenedor sin deformarla.

Por ejemplo, si tienes el siguiente CSS:



.universidad {

  background-image: url("universidad.jpg");

  background-size: cover;

  width: 100%;

  height: 200px;

}

Saludos!
